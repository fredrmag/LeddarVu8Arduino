Introduction

Library for the Leddar Vu8 LiDAR sensor (using SPI) made by LeddarTech. User guide can be downloaded here.

Written by Fredrik Magnussen for PSS-GEO AS.

The SPIdriver class is modified from the Arduino SD-card library SdFat written by William Greiman.
